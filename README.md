Atlas Motors is a modern car encyclopedia and comparison tool focused on cars from the 2000s to today.

Features
🔍 Global search by make, model, and year
🚗 Car cards with images, specs and quick stats
🧮 Compare up to 3 cars side‑by‑side
🤖 “Atlas AI” helper that gives a simple opinion based on speed, daily use and cost
🏷 Category filters: cheap, fast, daily, electric
🏭 Brand filter and year range sliders
🌗 Dark / light mode toggle
📱 Responsive layout for desktop and mobile
Tech
Pure HTML, CSS and JavaScript
No backend required for the demo
Car data stored in a JavaScript array (can be replaced with a real API or database later)
Roadmap
Connect to a real car database or API
Add more brands, trims and regional variants
User accounts and saved favourites
More advanced AI recommendations
